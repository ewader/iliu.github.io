宝宝七个月一天的辅食食谱

2010-01-21 人间世

颖颖七个月宝宝食谱（调整后）

一、主食

  • 母乳：全天多次喂养
  • 牛奶：上午 10:00 喂
  • 不建议：豆浆（易胀气，不易消化）

二、餐次安排

时间 食物
06:00 鸡蛋羹 + 母乳
10:00 牛奶
14:00 母乳
18:00 母乳
22:00 母乳

三、辅助食物

  1. 水/果汁：任选 1 种，每次 110ml，下午 15:00 喂
  2. 浓缩鱼肝油：每日 2 次，每次 2～3 滴
  3. 菜泥、碎菜、米粉、肝泥：下午 16:00，加入米粉 1～2 汤匙
  4. 馒头干、饼干：让宝宝自行啃咬，锻炼咀嚼能力，有助牙齿生长

四、推荐辅食做法

1. 面类

  • 金针菇面：龙须面、金针菇、虾仁、青菜、葱、油、盐、香油
  • 煮挂面：挂面、猪肝、虾肉、菠菜、蛋黄、肉汤、酱油
  • 蒸鸡肉鸡蛋面：挂面、蛋、高汤、青菜、胡萝卜、盐、酱油，蒸 15 分钟

2. 汤类

  • 猪骨高汤：煮 3 小时，撇去浮油，可用于冲米糊、泡面包、下面条
  • 豆腐蛋汤：蛋黄 + 海味汤 + 豆腐
  • 西红柿猪肝汤：猪肝 + 西红柿 + 葱 + 盐
  • 猪肝汤、三色肝末汤：猪肝搭配土豆泥、胡萝卜、西红柿、菠菜、洋葱，营养丰富，补铁、防贫血

3. 糊类

  • 鳕鱼苹果糊：鳕鱼 + 苹果
  • 南瓜洋葱糊：南瓜 + 洋葱水
  • 什锦豆腐糊：豆腐 + 胡萝卜末 + 鸡蛋 + 肉汤
  • 西红柿鱼糊、白萝卜鱼糊：鱼 + 西红柿 / 白萝卜 + 汤
  • 南瓜豆腐糊、鸡肝糊、蛋黄豌豆糊：提供钙质、蛋白质和卵磷脂
  • 其他泥类：奶酪红薯泥、土豆泥、海米蔬菜泥、茄泥、香蕉蒸、番茄豆腐泥、蔬菜豆腐泥、豆腐羹、猪肉豆腐羹、鱼泥、豆腐鸡蛋羹、鸡肉土豆泥、猪肝泥、鲜虾肉泥、火腿土豆泥、肝泥肉泥

4. 粥类

  • 牛奶粥：大米 + 牛奶粉
  • 蛋黄粥：大米 + 蛋黄
  • 青菜粥：大米 + 青菜，富含维生素 C、E
  • 水果麦片粥、米粉粥、鸡肉粥、草莓麦片粥：营养均衡
  • 面包粥、菜花蛋黄粥、小鱼蛋花粥、肝泥粥、花生（芝麻）粥、枣泥花生粥、香菇鸡粥、肝黄粥、鸡肉末碎菜粥、鱼肉松粥、蛋粥：多样选择，补充蛋白质、铁、钙和维生素

五、营养提示

  1. 蛋清：7 个月宝宝暂不食用，易过敏
  2. 猪肝、鸡肝：蒸或煮，去筋膜、脂肪，不易炒
  3. 米、菜：煮至软烂、易消化
  4. 辅食量：逐步增加，每次少量，观察宝宝耐受
  5. 均衡饮食：主食 + 辅食 + 母乳/牛奶，保证蛋白质、铁、钙、维生素
