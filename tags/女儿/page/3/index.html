女儿

相当想念女儿

记得一个同事曾经说过，出差在外，最思念的不是别人，是自己的女儿！当初的时候体会不到，现在真的体会到了这话的含义。 女儿岛11月11号就满5个月了，我也离开女儿有12天的时间了，想必女儿已经忘记了还有我这么一个老爸了吧。因为上次去赤峰出差了5天，回来的时候女儿就不认识我了。可是老爸却是每一天都把你放在心上。 早上醒来，会想女儿该醒了，该起床了吧，吃了多少蛋黄，看到满桌的饭菜是不是又流下了哈喇子呢。中午会想女儿今天吃的牛奶多不多，精神好不？晚上会想，小家伙该睡着了吧。有没有蹬被子？别人这丫头小，蹬被子的功夫却是一流的，两条腿向上一翘，等落下来的时候，被子也跑了。 想念女儿的依依呀呀，甚至女儿的哭闹都在想念，更别说这丫头的一颦一笑。何时我才能回去抱一抱我的宝儿呢？

致女儿
