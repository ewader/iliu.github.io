## Linux 删除文件夹和文件的命令

<span style="color: rgb(0, 0, 0);">删除文件夹实例：</span>

<span style="color: rgb(0, 0, 0);">rm -rf /var/log/httpd/access</span>

<span style="color: rgb(0, 0, 0);">将会删除/var/log/httpd/access目录以及其下所有文件、文件夹</span>

<span style="color: rgb(0, 0, 0);">删除文件使用实例：</span>

<span style="color: rgb(0, 0, 0);">rm -f /var/log/httpd/access.log</span>

<span style="color: rgb(0, 0, 0);">将会强制删除/var/log/httpd/access.log这个文件</span>

## 更改网站目录权限

网站根目录权限遵循：

文件644&nbsp;文件夹755&nbsp;&nbsp;权限用户和用户组www，其余文档777权限是不正常的

如出现文件权限问题时，请执行下面3条命令：

<pre class="ql-syntax" spellcheck="false"><span class="hljs-keyword">chown</span> -R www.www /data/wwwroot/
find /data/wwwroot/ -type d -<span class="hljs-keyword">exec</span> <span class="hljs-keyword">chmod</span> <span class="hljs-number">755</span> {} \;
find /data/wwwroot/ -type f -<span class="hljs-keyword">exec</span> <span class="hljs-keyword">chmod</span> <span class="hljs-number">644</span>
</pre>

                ## Linux 中 cp 命令（文件复制）
                
                1、cp fileOne fileTwo&nbsp;或者&nbsp;cp -i fileOne fileTwo&nbsp;&#8212;&#8212;&#8211;将文档One复制并命名成Two
                
                2、cp -i fileOne dirOne&nbsp;或者&nbsp;cp fileOne dirOne&nbsp;&nbsp;&#8212;&#8212;&#8212;&#8212;&#8212;将文档One复制到DirOne目录下
                
                3、cp -r dirOne dirTwo&nbsp;&nbsp;&nbsp;&#8212;&#8212;&#8212;&#8211;将目录dirOne复制到dirTwo下
                
                4、cp -r dirOne/\*.\*&nbsp;dirTwo&nbsp;&nbsp;&nbsp;&#8212;&#8212;&#8212;-将dirOne下所有文件包括文件夹复制到dirTwo
